Welcom bij Whack a Mole

  1. sla de mollen die omhoog komen
  2. sla zo veel mogelijk molen voordat de tijd op is
  3. als je een mol slaat krijg je 1 punt
  4. als je de mol niet optijd slaat krijg je geen punt
  5. als je mis slaat krijg je ook geen punt
  6. je kan dit niet met meerdere spelers spelen
